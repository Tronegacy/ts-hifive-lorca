## Getting Started

```bash
$ git clone https://github.com/mpppk/ts-hifive-lorca
$ cd ts-hifive-lorca
$ git submodule init
$ git submodule update
$ cd ts-hifive-boilerplate
$ npm install
$ npm run build
$ cd ..
$ make build
$ releng/build-{TARGET_OS}.sh
```
